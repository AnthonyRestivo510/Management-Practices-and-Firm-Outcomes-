# Management-Practices-and-Firm-Outcomes-
This project comes from the data set of Macchiavello and Morjaria focusses on ownership changes and the activity within the firm in Rwanda's coffee industry. Reporting how domestic and foreign acquisitions compare through multiple factors. Ultimately, the project highlights data cleaning, visualization and reproducible analysis through RMarkdown.
